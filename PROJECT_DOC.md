# hr-automation-android 项目文档

## 项目概述

本地 LLM + Embedding 驱动的智能招聘系统 Android 客户端，专注于评估模型在嵌入式设备（Android）上的使用效果。

## 模型选择与加载

### LLM 模型 (GGUF)

| 模型 | 大小 | 内存需求 | 适用场景 |
|------|------|----------|----------|
| Qwen2.5-0.5B-Instruct-Q4_0 | ~400MB | <2GB | 超轻量级，中文优化 |
| Phi-2-Q4_0 | ~500MB | <2GB | 英文推理优秀 |
| SmolLM2-1.7B-Instruct-Q4_0 | ~1GB | ~2GB | 性能均衡 |
| Qwen2-0.5B-Instruct-Q4_0 | ~420MB | <2GB | Qwen2 基础版 |
| TinyLlama-1.1B-Chat-Q4_K_M | ~670MB | ~3GB | 生态丰富 |
| Gemma-2B-Q4_K_M | ~1.6GB | ~3GB | Google Gemma |
| StableLM-3B-Q4_K_M | ~1.9GB | ~4GB | 长上下文支持 |

### Embedding 模型 (ONNX)

| 模型 | 大小 | 向量维度 | 适用场景 |
|------|------|----------|----------|
| all-MiniLM-L6-v2 | ~90MB | 384 | 英文文本相似度（资源受限首选） |
| all-MiniLM-L12-v2 | ~120MB | 384 | 英文文本相似度（更高精度） |
| paraphrase-MiniLM-L6-v2 | ~90MB | 384 | 英文改写检测 |
| bge-small-en-v1.5 | ~130MB | 384 | 英文文本召回 |
| bge-base-zh-v1.5 | ~170MB | 768 | 中文文本召回（推荐） |
| bge-large-zh-v1.5 | ~500MB | 1024 | 中文高精度语义匹配 |

## 可行的优化方向

### 1. 模型推理优化

| 优化方向 | 描述 | 预期效果 |
|----------|------|----------|
| **GPU 加速** | 使用 Android NNAPI 加速模型推理 | 推理速度提升 2-5 倍 |
| **模型量化** | 尝试更激进的量化 (Q2, Q3) | 模型体积减小 30-50% |
| **模型缓存** | 预加载常用模型到内存 | 冷启动时间减少 |
| **增量推理** | 只推理变化的部分 | 响应时间减少 |

### 2. 用户体验优化

| 优化方向 | 描述 | 预期效果 |
|----------|------|----------|
| **智能推荐** | 基于历史行为推荐候选人/职位 | 匹配效率提升 |
| **批量操作** | 支持批量发送消息、批量标记 | 操作效率提升 |
| **快捷搜索** | 全局搜索入口，快速定位 | 使用便捷性提升 |
| **暗色模式** | 支持深色/浅色主题切换 | 用户体验优化 |

### 3. 数据处理优化

| 优化方向 | 描述 | 预期效果 |
|----------|------|----------|
| **本地向量库** | 使用 FAISS 或 Annoy 做高效向量检索 | 匹配速度提升 10x |
| **增量同步** | 只同步变化的数据 | 网络流量减少 80% |
| **数据压缩** | 压缩存储的画像和向量数据 | 存储空间减少 50% |

### 4. 企业级功能

| 优化方向 | 描述 | 预期效果 |
|----------|------|----------|
| **多语言支持** | 支持中英文界面切换 | 适用更多用户 |
| **权限管理** | 不同角色不同权限 | 团队协作支持 |
| **API 集成** | 支持对接第三方招聘系统 | 数据互通 |
| **数据备份** | 云端备份和恢复 | 数据安全 |

### 5. AI 能力增强

## 技术架构

```
┌─────────────────────────────────────────────┐
│                   UI Layer                   │
│  MainActivity + Fragments (ViewBinding)     │
│  Jobs / Candidates / Matches / Settings     │
│  MatchingCenter (ViewPager2 + TabLayout)    │
├─────────────────────────────────────────────┤
│                Data Layer                    │
│  Room Database (4 entities, 4 DAOs)         │
│  AppDatabase (singleton, thread-safe)       │
├─────────────────────────────────────────────┤
│                 ML Layer                     │
│  LocalLLMService    → llama.cpp JNI         │
│                     → 远程 Ollama            │
│  LocalEmbeddingService → ONNX Runtime       │
│  cosineSimilarity → 向量匹配计算             │
├─────────────────────────────────────────────┤
│              Networking                      │
│  Retrofit 2.11 + OkHttp 4.12 + Gson         │
└─────────────────────────────────────────────┘
```

## 数据模型

### JobEntity
| 字段 | 类型 | 描述 |
|------|------|------|
| id | Long | 主键 |
| title | String | 职位名称 |
| requirements | String | 职位要求 |
| status | String | 状态 (active/closed/archived) |
| profile | String | AI 生成的职位画像 (JSON) |
| createdAt | Long | 创建时间 |

### CandidateEntity
| 字段 | 类型 | 描述 |
|------|------|------|
| id | Long | 主键 |
| name | String | 姓名 |
| email | String | 邮箱 |
| phone | String | 电话 |
| resume | String | 简历内容 |
| profile | String | AI 生成的候选人画像 (JSON) |
| createdAt | Long | 创建时间 |

### MatchEntity
| 字段 | 类型 | 描述 |
|------|------|------|
| id | Long | 主键 |
| jobId | Long | 职位 ID |
| candidateId | Long | 候选人 ID |
| score | Float | 综合匹配度 |
| skillMatch | Float | 技能匹配度 |
| experienceMatch | Float | 经验匹配度 |
| educationMatch | Float | 学历匹配度 |
| matchReason | String | 推荐理由 |
| status | String | 状态 (pending/screened/accepted/rejected) |
| evaluation | String | AI 评估内容 |
| createdAt | Long | 创建时间 |

### ApplicationEntity
| 字段 | 类型 | 描述 |
|------|------|------|
| id | Long | 主键 |
| jobId | Long | 职位 ID |
| candidateId | Long | 候选人 ID |
| coverLetter | String | 求职信 |
| status | String | 状态 (pending/accepted/rejected) |
| appliedAt | Long | 投递时间 |

## 构建与运行

### 环境要求

- Android Studio Hedgehog (2023.1.1) 或更新
- JDK 17
- Android SDK：compileSdk 35, minSdk 26
- Gradle 8.4

### 构建命令

```bash
# Debug APK
./gradlew assembleDebug

# Release APK
./gradlew assembleRelease

# 运行测试
./gradlew test

# Lint 检查
./gradlew lint
```
